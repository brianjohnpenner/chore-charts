Quick start

  1. Set the child's name and page orientation up top. Add more kids with Add Child.
  2. Tweak the Days — rename a day or tap its color swatch to recolor that column.
  3. In each section (Morning, Daytime, …), tap Add Chore. Pick an icon, type a name, and check the days it applies to.
  4. Toggle the to show or hide a label, and the coin to mark a chore as paid.
  5. Drag the handle to reorder chores or whole sections.
  6. Use Preview to see the finished chart, then Print Current or Print All.

How your data is stored

  • Your charts save automatically in this browser on this device. Nothing is uploaded to a server and there's no sign-in.
  • Because it lives in this browser, clearing site data or switching devices/browsers means it won't follow you — use the buttons below to keep a copy.
  • Save / Open JSON keeps a chart as a file you control. Download and Share JSON hand a copy to someone else or another device. Import loads one back in.
  • Reset clears everything back to the sample chart.
  • Want it handy and offline? Install the app to your home screen when prompted.
